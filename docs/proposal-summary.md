# ControlSense AI proposal summary

ControlSense AI is a local-first educational app for classical control theory, modern robotic control, embodied AI, Physical AI and reinforcement learning. It turns the supplied proposal into a working MVP: lessons, simulations, low-stakes mini-tests, local progress tracking and a bounded tutor called LoopCoach.

The implementation is intentionally dependency-free for the first build, so it can run locally on macOS, Windows and Linux through any modern browser and later be wrapped in Tauri for desktop distribution.
